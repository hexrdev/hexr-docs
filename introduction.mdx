## From decorator to production in three commands

<Steps>
  <Step title="Write your agent" icon="code">
    Use any Python framework — CrewAI, LangChain, OpenAI, or raw Python. Add one decorator.
    ```python
    from hexr import hexr_agent, hexr_tool, hexr_llm

    @hexr_agent(name="research-analyst", tenant="acme-corp")
    def analyze(topic: str):
        s3 = hexr_tool("aws_s3")            # authenticated — no keys in code
        client = hexr_llm(openai.OpenAI())   # auto-traced with cost attribution
        return client.chat.completions.create(
            model="gpt-4o",
            messages=[{"role": "user", "content": f"Analyze {topic}"}]
        ).choices[0].message.content
    ```
  </Step>
  <Step title="Build + Push" icon="hammer">
    The CLI scans your source code via AST, discovers every `@hexr_agent`, `hexr_tool()`, and `hexr_llm()` call, then generates a Dockerfile, Kubernetes manifests, and injects four sidecar containers automatically.
    ```bash
    hexr build   # AST discovery → Dockerfile + manifests + sidecar injection
    hexr push    # Container image → registry
    ```
  </Step>
  <Step title="Deploy" icon="rocket">
    One command schedules your agent as a Kubernetes Pod. The Auto-Registrar creates a SPIFFE identity. The Credential Injector provisions cloud access. Envoy starts mTLS. Observability flows.
    ```bash
    hexr deploy  # Kubernetes scheduling → SPIFFE registration → ready
    ```
  </Step>
</Steps>

## What Hexr gives you — automatically

Every deployed agent gets all of this with zero configuration:

<CardGroup cols={2}>
  <Card title="🔐 Cryptographic Identity" icon="fingerprint">
    Every agent **process** gets a unique SPIFFE X.509 certificate — not just the container. Mutual TLS everywhere. Certificates rotate automatically.
  </Card>
  <Card title="🔑 Zero-Secret Cloud Access" icon="key">
    `hexr_tool("aws_s3")` returns an authenticated boto3 client. JWT-SVID → AWS STS / GCP WIF / Azure exchange. Three-tier cache delivers sub-ms latency.
  </Card>
  <Card title="📊 Full Observability" icon="chart-mixed">
    `hexr_llm()` wraps any LLM client with OpenTelemetry spans. Per-agent cost attribution, token counts, latency histograms — all correlated by SPIFFE identity.
  </Card>
  <Card title="🔒 Zero-Trust Networking" icon="shield-halved">
    Envoy sidecar handles all traffic. Every connection is mTLS. OPA policy enforcement at every service boundary. No plaintext, ever.
  </Card>
  <Card title="📡 Agent-to-Agent Protocol" icon="messages">
    Built-in A2A sidecar with JSON-RPC 2.0. Agent discovery via Agent Cards, task lifecycle, SSE streaming — all authenticated via SPIFFE.
  </Card>
  <Card title="🔧 Secure Tool Access" icon="wrench">
    Import any OpenAPI spec as MCP tools. Gateway handles credential injection from Vault automatically. Sandbox runs code in Firecracker microVMs.
  </Card>
</CardGroup>
