## Usage

```bash
hexr cache <subcommand>
```

---

## Subcommands

### hexr cache status

Show current cache state:

```bash
$ hexr cache status

Hit Rates (last 1h):

  L2 (Valkey):   18.4%

```

### hexr cache clear

Clear all cached credentials:

```bash
$ hexr cache clear

✓ L1 cache cleared (in-memory)
✓ L2 cache cleared (Valkey)

Note: Next hexr_tool() call will perform a full credential exchange.
```
