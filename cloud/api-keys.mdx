## Generating Keys

1. Open [app.hexr.cloud](https://app.hexr.cloud)
2. Navigate to **API Keys**
3. Click **Generate New Key**
4. Copy the key — it will only be shown once

API key format: `hxr_live_<64 hex characters>`

---

## Using Keys

### CLI Authentication

```bash
hexr login --key hxr_live_0f0ea94b...
```

### API Authentication

```bash
curl -H "Authorization: Bearer hxr_live_0f0ea94b..." \
     https://api.hexr.cloud/api/v1/tenants
```

---

## Key Permissions

| Permission | Can Do |
|-----------|--------|
| **Admin** | Everything: manage tenants, users, keys, deploy agents |
| **Deploy** | Build, push, deploy agents. Cannot manage users or keys |
| **Read-only** | View agents, metrics, logs. Cannot modify anything |

---

## Security Best Practices

<Warning>
  - Never commit API keys to source control
  - Rotate keys regularly
  - Use the minimum permission level needed
  - Revoke keys immediately when team members leave
</Warning>

Store keys securely:

```bash
# macOS Keychain
security add-generic-password -s "hexr-api-key" -a "hexr" -w "hxr_live_..."

# Environment variable (development only)
export HEXR_API_KEY="hxr_live_..."
```
