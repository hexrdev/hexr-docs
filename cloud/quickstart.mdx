## What Is Hexr Cloud?

Hexr Cloud is the fully managed deployment of the Hexr platform. We handle:

- ✅ SPIRE Server + Agent (identity)
- ✅ Credential Injector (cloud access)
- ✅ Gateway, Vault, Sandbox (services)
- ✅ Observability stack (Grafana, Prometheus)
- ✅ Dashboard (management UI)

You bring your agent code — we handle everything else.

---

## 1. Get Access

<Tabs>
  <Tab title="Via Dashboard">
    1. Go to [app.hexr.cloud/onboard](https://app.hexr.cloud/onboard)
    2. Enter your **organization name**, **email**, and **invite code**
    3. Click **Create Account**
    4. **Save your API key** — it's shown only once (`hxr_live_...`)
  </Tab>
  <Tab title="Via CLI">
    ```bash
    curl -s -X POST \
      -H "Content-Type: application/json" \
      -d '{
        "name": "your-org-name",
        "contact_email": "you@example.com",
        "invite_code": "YOUR_INVITE_CODE"
      }' \
      https://api.hexr.cloud/v1/tenants/onboard | python3 -m json.tool
    ```
    Save the `api_key` from the response.
  </Tab>
</Tabs>

<Info>
  Don't have an invite code? Request early access at [hexr.dev](https://hexr.dev).
</Info>

---

## 2. Install and Authenticate

```bash
pip install "hexr-sdk[cli]" --extra-index-url https://pypi.hexr.cloud/simple/
hexr login --key hxr_live_your_api_key_here
```

---

## 3. Build and Deploy

```bash
# Analyze your agent
hexr build my_agent.py --tenant your-tenant

# Build and push (uses Google Cloud Build — no local Docker needed)
hexr push --cloud --tenant your-tenant

# Deploy to Hexr Cloud
hexr deploy .hexr --cloud
```

---

## 4. Monitor

Open [app.hexr.cloud](https://app.hexr.cloud) to:

- View deployed agents and their status
- See traces for every LLM and tool call
- Track costs per agent, per model
- Manage API keys and team members
